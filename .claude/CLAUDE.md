# FluxStudio

Creative collaboration platform for teams.

## Quick Start
```bash
npm install
npm run dev
```

## Architecture
- **Frontend**: React with modern state management
- **Backend**: Node.js/Express
- **Database**: PostgreSQL
- **Deployment**: DigitalOcean App Platform

## Project Structure
- `apps/` - Application packages (monorepo)
- `packages/` - Shared packages
- `src/` - Source code
- `.fluxstudio/` - FluxStudio-specific configs

## Key Features
- Real-time collaboration
- Creative workflow tools
- Team management
- Asset management

## Development Guidelines
- Follow monorepo conventions if applicable
- Use environment-specific configs
- Test with both local and production databases
- Follow established component patterns

## Deployment
- Production: DigitalOcean App Platform
- Preview deploys via GitHub integration
