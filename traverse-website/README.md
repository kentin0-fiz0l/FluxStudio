# Traverse Arc

**Personal Flight, Redefined**

A consumer-owned, autonomous, electric personal air vehicle that fits into everyday life — making distance irrelevant.

## About

Traverse Arc is building a new category of personal transportation: consumer-owned, autonomous, electric personal air vehicles that fit into existing urban life as naturally as a car.

### Arc S2 - The Flagship

Arc S2 is a two-seat, side-by-side consumer sport vehicle that is:
- Fully electric and zero-emission
- Autonomous by design (no pilot license required)
- Small enough to fit in a standard parking space
- Designed to live in a garage

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** - Fast build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
